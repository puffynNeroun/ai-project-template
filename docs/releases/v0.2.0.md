# Forge v0.2.0 Release Notes

## Status

Prepared release notes for the future Forge v0.2.0 release.

This document does not create the Git tag, GitHub release, or package publication.

## Summary

Forge v0.2.0 turns the project template from a mostly document-driven workflow into a controlled local lifecycle CLI.

The main improvement is that common workflow steps are now executable, test-covered, and validated by the Forge contract system.

## Highlights

- Added lifecycle status reporting.
- Added task scaffolding.
- Added stage transition command.
- Added artifact report scaffolding.
- Added task completion command.
- Added full workflow smoke validation.
- Expanded the Forge Validator test baseline to 203 passing tests.
- Preserved contract validation through CI.

## Added Commands

### Status

pnpm -C tools/forge-validator run status

Shows the current Forge lifecycle state:

- current Git branch
- working tree state
- active task
- task board state
- required artifact presence
- stale verification text scan

### Create Task

pnpm -C tools/forge-validator run task:new -- --id TASK-XXXX --title "Task title"

Creates a new task contract and updates docs/TASKS.md.

### Move Task Stage

pnpm -C tools/forge-validator run task:stage -- --id TASK-XXXX --stage planner

Supported stages:

- planner
- builder
- tester
- reviewer

### Create Artifact Report

pnpm -C tools/forge-validator run artifact:new -- --id TASK-XXXX --type plan

Supported artifact types:

- plan
- build_report
- test_report
- review_report

### Complete Task

pnpm -C tools/forge-validator run task:complete -- --id TASK-XXXX

Marks a ready_for_pr task as completed and updates docs/TASKS.md.

### Workflow Smoke Test

pnpm -C tools/forge-validator run workflow:smoke

Runs a full local lifecycle in a temporary fixture repository:

1. planner stage
2. plan artifact
3. builder stage
4. build_report artifact
5. tester stage
6. test_report artifact
7. reviewer stage
8. review_report artifact
9. task completion
10. final board and task contract assertions
